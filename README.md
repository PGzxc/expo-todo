## 二 开发环境

* 操作系统：Windows 11 专业版 22H2
* Node：v18.18.2，node源：taobao
* Yarn：1.22.19
* jdk：openjdk 11.0.19 2023-04-18
* Android SDK Build-Tools：33.0.0
* react-native：0.72.6

## 三 如何试用

### 3.1 下载项目

```
git clone https://github.com/PGzxc/expo-todo.git
```

### 3.2 安装依赖

进入到项目根目录，打开cmd终端，执行如下指令

```
yarn
```

### 3.3 运行项目

```
npx expo start
```
